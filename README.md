# CulturAll
CulturAll is a web project designed to help users discover cultural, social, and leisure activities in a city. The platform organizes different types of activities into thematic categories, making it easier for users to explore places and events based on their interests.

The website currently consists of a multi-page frontend built using HTML, CSS, and JavaScript. Each section of the site represents a different category such as Art, Sports, and Eat & Drink, allowing users to navigate between pages and explore various activities.

The project includes separate homepage versions for authenticated and non-authenticated users, demonstrating the concept of user-based navigation and interface structure.

Features

Multi-page website structure

Category-based navigation

Responsive interface

Organized frontend structure (HTML, CSS, JavaScript)

Technologies Used

HTML5

CSS3

JavaScript

Project Status

The project is currently in the frontend stage. Future improvements may include backend integration, user authentication, and a database for managing events and activities.
